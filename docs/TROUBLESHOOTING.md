# TROUBLESHOOTING.md
Как устранить проблемы в SM Codex Portal

Документ для разработчиков и тимлидов, содержит быстрые инструкции по устранению типичных ошибок.

---

# 1. Проблема: Страница не загружается

### Возможные причины:
- ошибка в Router  
- неверный путь в import()  
- ошибка в инициализации модуля

### Проверка:
1. Консоль DevTools  
2. Найти ошибку `Failed to fetch module`  
3. Проверить путь в Router  
4. Проверить правильность имени файла  

---

# 2. Проблема: Поп-ап калькулятора не открывается

### Причины:
- отсутствует `data-calc-open`
- ошибка в `templates.js`
- ошибка в `init*Calculator()`

### Решение:
1. Проверить имя калькулятора  
2. Проверить шаблон  
3. Проверить логи DevTools  

---

# 3. Проблема: Квиз не запускается

### Причины:
- неверный `data-quiz-category`
- JSON не загружается
- ошибка в registry.js

### Решение:
1. Проверить JSON  
2. Проверить ключ в реестре  
3. Проверить popup.js  

---

# 4. Проблема: Sticky-панели не работают

### Причины:
- конфликт CSS  
- неверный z-index  
- неправильная вложенность разделов

### Решение:
1. Проверить контейнеры  
2. Проверить структуру секций  
3. Проверить стили  

---

# 5. Проблема: калькулятор считает неправильно

### Причины:
- цифры парсятся неправильно  
- логика в init* искажена  
- шаблон использует старые ID

### Решение:
1. Проверить ID входных полей  
2. Проверить формулы  
3. Проверить core.js
